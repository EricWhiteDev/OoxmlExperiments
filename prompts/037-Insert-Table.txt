Make a new button in the task pane after the addHtml button.  Name this button addTable.  When the user clicks this button, add a table at the end of the selection.  As necessary, break the paragraph that contains the selection, and add the table in the middle of the paragraph.  If the selection is between paragraphs, then add the table at that location.

Here is the table data:

const tableData = [
        ["Name", "ID", "Birth City"],
        ["Bob", "434", "Chicago"],
        ["Sue", "719", "Havana"],
    ];

use the paragraph.insertTable to add the table that contains the specified tableData.
