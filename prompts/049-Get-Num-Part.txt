To the right of the "Get Style Def Part", add a new button, "Get Num Part"

When click do:

- get the entire body of the document
- Open with OpenXmlSdkTs using the WmlPackage class.
- Get the mainDocument part.

- Get the XDocument.
- format it with indentation using LtXmlTs functionality that serializes with indenting, and then set textDisplay to the XML string. set textViewerTitle to "Numbering Part"
